# Ballet History 🩰
> Projeto de Pesquisa e Inovação | SPTech School | 1º Semestre

## Sobre o Projeto
O **Ballet History** é uma plataforma interativa que preserva a memória do ballet clássico. Além de ser um repositório de histórias detalhadas, o projeto foca na experiência do usuário através da gamificação e análise de dados, transformando a leitura em uma jornada de evolução profissional dentro da hierarquia da dança.

## Funcionalidades Principais
- **Conteúdo Imersivo:** Histórias detalhadas de ballets de repertório com.
- **Área Restrita (Login/Cadastro):** Acesso exclusivo a funcionalidades interativas e personalizadas.
- **Quizz de Performance:** Avaliação de conhecimento sobre as obras. O desempenho do usuário define seu "Cargo" na companhia (de Corpo de Baile a Bailarino Principal), com dados persistidos em banco.
- **Quizz de Perfil:** Algoritmo que define qual personagem e variação clássica melhor se adaptam à personalidade do usuário.
- **Dashboard:** Visualização de dados de desempenho e estatísticas da comunidade utilizando gráficos dinâmicos.
